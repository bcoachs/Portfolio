# Regeln und Grenzwerte

Diese JSON-Dateien legen die Standardregeln und Zielkorridore für dein Dividendenportfolio fest. Du kannst sie im Admin‑Modus anpassen.

- `default_rules.json`: Enthält Zielgewichte pro Rolle, zulässige Dividendenrenditen, Dividendenwachstumsraten und KPI‑Schwellen.
- `kpi_thresholds`: Grenzwerte für Free‑Cashflow‑Payout‑Ratio, Gewinn‑Payout‑Ratio, Verschuldung und Kapitalrendite.

Ändere die Werte entsprechend deiner persönlichen Anlagestrategie. Über die Admin‑Seite im Tool werden die aktuellen Regeln geladen und versioniert gespeichert.
