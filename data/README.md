# Daten

In diesem Ordner legst du deine Portfolio‑Daten ab. Zum Start gibt es eine Beispiel‑CSV (`sample_portfolio.csv`) mit der richtigen Spaltenstruktur:

- `Ticker`: Börsenticker der Aktie
- `Name`: Unternehmensname
- `Rolle`: Zielrolle im Portfolio (z. B. "Freiheitsmotor", "Wachstumsdividenden")
- `Stueck`: Anzahl der gehaltenen Stücke
- `Einstandskurs`: Kaufkurs je Stück
- `Marktwert`: Aktueller Gesamtwert der Position
- `Waehrung`: Währung des Titels
- `Dividendenrendite`: Erwartete Dividendenrendite (dezimal, z. B. 0.03 für 3 %)
- `Dividendenwachstum`: Erwartetes Dividendenwachstum (rollierendes 5‑Jahres‑CAGR)
- `ROIC`: Return on Invested Capital
- `Payout_FCF`: Free‑Cashflow‑Payout‑Ratio
- `NetDebt_EBITDA`: Verschuldung in Relation zum EBITDA
- `Interest_Coverage`: Zinsdeckungsgrad

Du kannst deine eigenen CSV‑Daten aus Parqet exportieren und im Tool hochladen.
